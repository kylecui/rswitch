## 如何贡献翻译

1. Fork 本仓库
2. 在 `docs/zh-CN/` 目录下创建翻译文件
3. 翻译完成后更新本索引中对应文档的翻译状态为 ✅ 已完成
4. 提交 Pull Request

**翻译规范**:
- 技术术语保留英文（如 XDP、BPF、CO-RE、AF_XDP、VLAN、QoS）
- 代码块和命令行不翻译
- 保持与英文原文相同的文档结构
- 文件名使用英文，与原文保持一致
