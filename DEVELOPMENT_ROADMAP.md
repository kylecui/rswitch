└─────────────────────────────────────────────────────────────────────────────┘
  🚧 stp.bpf.c             - 生成树协议
     │

     ├─ 功能: BPDU 处理, 端口状态机, 角色选举
     ├─ 状态: Blocking → Listening → Learning → Forwarding
     ├─ 难度: 高
     ├─ 工作量: 3-4 周
     ├─ 优先级: P3 - 低
     └─ 挑战: 定时器实现（需用户空间辅助）
  
  🚧 lacp.bpf.c            - 链路聚合
     │
     ├─ 标准: IEEE 802.3ad LACP
     ├─ 功能: LACPDU 处理, 链路检测, 负载均衡
     ├─ 负载均衡: MAC/IP/五元组
     ├─ 难度: 高
     ├─ 工作量: 3-4 周
     └─ 优先级: P3 - 低
  
  🚧 lldp.bpf.c            - 拓扑发现
     │
     ├─ 标准: IEEE 802.1AB LLDP
     ├─ 功能: LLDP 帧发送/接收, 邻居信息维护
     ├─ TLV: Chassis ID, Port ID, TTL, System Name/Description
     ├─ 难度: 低
     ├─ 工作量: 1 周
     └─ 优先级: P3 - 低
  
  📅 时间线:
     05-06 月: STP/RSTP
     06-07 月: LACP
     07-08 月: LLDP
     09-10 月: v2.0 集成测试
