# 参考型文档写法

## 配置文档

对每个配置项尽量说明：

- 名称
- 类型
- 是否必填
- 默认值
- 示例
- 风险影响 / 注意事项

## CLI 文档

对每个命令尽量说明：

- 命令用途
- 参数
- 示例
- 成功输出
- 失败情形
- 返回码（如可确认）

## API 文档

对每个接口尽量说明：

- 路径 / 方法
- 功能
- 请求参数
- 请求体
- 认证要求
- 返回结果
- 错误响应
- 示例调用

## SDK / library 文档

优先围绕 public surface：

- 安装
- 初始化
- 最小调用示例
- 常见调用路径
- 错误处理
- 资源释放 / 生命周期
