# 文档集蓝图

## 最小可用文档集

适合大多数项目的默认组合：

- `README.md`
- `docs/quickstart.md`
- `docs/configuration.md`
- `docs/troubleshooting.md`

## 如果项目是 API 服务

增加：

- `docs/api.md`
- `docs/deployment.md`

## 如果项目是 CLI

增加：

- `docs/cli.md`
- `docs/examples.md`

## 如果项目是 SDK / library

增加：

- `docs/usage.md`
- `docs/public-api.md`

## 选择原则

不要默认全生成。  
优先覆盖：

1. 它是什么
2. 如何开始
3. 如何配置
4. 如何常用
5. 出问题怎么办
